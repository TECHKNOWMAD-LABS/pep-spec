Last updated: 2026-04-09

## Overview

PEP Spec offers flexible pricing models to serve developers, startups, and enterprises building autonomous Python systems.

---

## Free Tier (Open-Core)

Free for individual developers and small projects

### Included Features
- 7 standard PEPs (Organism, Judge, Engine, Event Log, Sharing, Privacy, Agent)
- Local development environment
- Community Discord support
- MIT License option for open-source redistribution
- Access to GitHub repository issues

### Limitations
- No cloud hosting
- No priority support
- Limited to 10 organisms

### Use Cases
- Side projects
- Learning & experimentation
- Small open-source tools

## FAQ

**Q: Can I switch plans?**  
A: Yes, anytime. Pro to Free is instant; Enterprise requires 30-day notice.

**Q: Do you offer payment plans?**  
A: Annual subscribers can pay in monthly installments (+₹5k fee).

**Q: Is there a minimum contract for Enterprise?**  
A: No minimum term. Yearly preferred for best pricing.

**Q: Do you offer API access?**  
A: Yes, included in Enterprise. Pro has limited API rate.

**Q: Can I use PEP Spec in my open-source project?**  
A: Yes, with MIT license option (free tier attribution required).
